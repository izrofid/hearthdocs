If you discover a bug while playing Pokemon Hearth you can report them in the [discord server](https://discord.gg/XDetmVQQZy). Please read further to learn how to do so.

## Before Reporting

<Steps>
1. **Test the bug**: Try to reproduce it consistently.

2. **Check for duplicates**: Make sure it hasn’t already been reported in the server.

3. **Use the bug report template**: Refer to the next message in the thread for the correct format.
</Steps>

<Aside icon="warning" type="caution" title="Warning">
##### Do Not Report Bugs If

- You’re not using the latest version of the game.

- You’re using external cheat codes or save editors.

- The issue is a temporary crash that disappears after a hard reset.

- You did not patch the game yourself.
</Aside>


### Template
Copy this template to make your bug reports. Each item is explained below.

<Card title="Bug Report Template" icon="pen">

- Bug Description:
- Cause:
- Cheats Used:
- Location:
- Does Saving/Reseting Fix The Bug:
- Pokemon:
- NPC:
- Emulator:
- Game Version:
- Can be reproduced:
- Screenshot / Video:

</Card>

### Description Of Each Bullet Point

- **Bug Description:**
Explain what unintended behavior/error/bug you found

- **Cause:**
 Explain what you've done to encounter this bug, the more detail the better

- **Cheats Used:**
Mention if you've used any cheats or not and what kind, such as gameshark/action replay cheats, or any other kind. <Badge text="Important" variant="note" />

- **Location:**
Indicate where you were in-game when the bug happened

- **Does Saving/Reseting Fix The Bug:**
Yes/No, please try, and please try with the in-game save instead of save states

- **Pokemon:**
 Indicate if a Pokemon was involved in the bug or not, and if so, which one
- **NPC:**
 Indicate if an NPC was involved in the bug or not, and if so, which one. For bugs involving trainers, please specify the trainer name.

- **Emulator:**
 Indicate what emulator is being used. <Badge text="Important" variant="note" />

- **Game Version:**
The Hearth version you're playing on

- **Can be reproduced:**
Yes/No, please try, and list steps if possible

- **Screenshot / Video:**
 Yes/No, please indicate if you sent one or not

Clear, detailed reports help the team identify and fix issues faster.
